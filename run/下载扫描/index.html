项目下载与警告扫描

PrepareProject

项目下载与扫描脚本，使用-h选项查看帮助，使用选项即执行相关命令，可同时执行多个。

短选项 长选项 含义
-c --collect 从Maven仓库中收集项目的版本数据等信息，保存到 project.json 文件
-d --download 下载jar包和源码jar包，并解压源码
-s --scan 扫描下载好的jar包
-u --update 整合警告并对相关java文件预处理
